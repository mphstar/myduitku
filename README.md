# MyDuitKu 💰

MyDuitKu adalah aplikasi pengelola keuangan pribadi yang dirancang untuk memudahkan pencatatan pemasukan dan pengeluaran sehari-hari. Dibangun menggunakan Flutter untuk performa tinggi dan tampilan antarmuka yang modern.

## 📱 Fitur Utama

- **Dashboard Interaktif**: Ringkasan keuangan dengan grafik dan "Action Cepat" untuk akses mudah.
- **Pencatatan Transaksi**: Catat pemasukan dan pengeluaran dengan kategori yang dapat disesuaikan.
- **Manajemen Anggaran (Budget)**: Tetapkan batas anggaran bulanan untuk mengontrol pengeluaran.
- **Laporan Komprehensif**: Analisis keuangan dengan filter tanggal (termasuk rentang tanggal kustom) dan visualisasi grafik.
- **Manajemen Data**:
  - **Export Data**: Backup data ke format JSON dan simpan langsung ke penyimpanan perangkat (File Manager).
  - **Import Data**: Restore data dari file backup JSON.
  - **Reset Data**: Opsi untuk menghapus seluruh data aplikasi dengan aman.
- **Kategori Kustom**: Tambah, edit, dan hapus kategori transaksi sesuai kebutuhan.
- **Mode Gelap/Terang**: Menyesuaikan dengan tema sistem.

## 🛠️ Teknologi yang Digunakan

- **Framework**: [Flutter](https://flutter.dev)
- **Bahasa**: Dart
- **Database Lokal**: [Hive](https://pub.dev/packages/hive) (Cepat, ringan, dan NoSQL)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **File Picker**: Integrasi native file picker untuk simpan/buka file backup.

## 🚀 Instalasi

1. **Clone repository ini:**
   ```bash
   git clone https://github.com/username/myduitku.git
   ```

2. **Masuk ke direktori project:**
   ```bash
   cd myduitku
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Jalankan aplikasi:**
   ```bash
   flutter run
   ```
